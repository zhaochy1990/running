# P1W2：基础重建第 2 周（5/11 — 5/17）

> **本周定位**：P1W1 跑量 122 km / 强度 3 次 / 最长 22K（计划 63 km / 1 次 / 22K），KPI 4/5 红灯。**P1W2 不是再写一份"更严的"plan，是承认现实做选项 B**：跑量 95-100 km，强度限 2 次/周（团队晚课 1 次替代节奏跑 + STRIDE 长距离 1 次），但**HR 上限不可商量**，**力量基线测试本周必做**。**5/17 末若 RHR 仍 ≥51 / fatigue ≥48 / Z3+Z4 占比 ≥30% → 强制走选项 A（退出团队晚课）**。

## 上周（P1W1）小结

- **跑量 122 km**（计划 63，+94%）/ **强度 3 次**（计划 1）：5/6 晚 8×400m HR 165+，5/8 晚 12×600m HR 160+，5/10 末 2K 冲到 HR 168
- **力量基线测试连续第二周未做**（5/5 跑了 20K + 6ST 替代）
- **5/10 长距离 22K**：前 20K HR 121→146 教科书级 Z2（漂移 16 bpm 偏高但可接受），**末 2K 冲刺 4:09→3:51 / HR 158→168**——把整次训练负荷从 ~120 推到 166
- **5/10 收盘**：fatigue 49 🟡 / ATI 133 🔴（一周从 117→133）/ CTI 119 / TL ratio 1.11 / state Very High **整周 7 天** / TSB -14 / RHR 51（基线 +4-9）/ VO2max 56 / 跑力 92.2

> 春训三连马 168→166→165 的同种模式：**前 30K 体感好就一路推上去**——P1W2 必须先打断这个。

## 跑步专项

**周二 5/12 — 轻松 12K + 4×100m strides**：5:30-6:30 出发；0-9K 稳 5:00-5:20/HR<142；9.5-10K 4×100m strides @ 3:30-3:40，组间走 60s；10-11K 收尾。**前提**：5/11 完全休息后跟腱无反应；力量测试还没做 → strides 改稳态 11K 也行。

**周三 5/13 — 力量基线 11 项 + 晚 8K 恢复**：测试详见 § 力量训练；测试后 4-6 小时再跑 8K @ HR<138 / 5:30+。**测试日不安排团队课**——本周必须确保测试落地。

**周四 5/14 — 力量 A + 轻松 10K（不双跑）**：力量 A 45 min 在晨；晚跑 10K @ HR<142 / 5:00-5:20。**不要早晚双跑**——P1W1 的 ATI 飙到 133 一半原因是双跑日累积。**HR avg 超 145 → 立即降到 5:30 走跑结合**。

**周五 5/15 — 团队课替代节奏跑（强度课 1/2，本周唯一一次）⚡**：
- 团队课如果是节奏/间歇/MP 段 → **HR 上限 167**（不是 170），单次 Z4-Z5 累积 ≤20 min，超出立即降到 4:30
- 团队课如果是 VO2max 间歇（如 1000m+） → 数量限 6-8 个，超出离场
- 团队课如果是轻松团跑（无质量段） → 本周 0 强度课 → 周六改节奏跑 6K @ 4:15-4:20 / HR 158-163（STRIDE 自主版）
- **失败止损**：第 2 个 rep HR>170 → 立即停下喝水 → 减半数量；第 3 个仍>170 → 终止改慢跑回家
- **绝不双跑**：5/15 团队课已是当日唯一 quality

**周六 5/16 — 恢复 8K**：5/15 后晨起跟腱 ≤1/10 + RHR ≤50 + 主观 ≤4 → 跑 8K @ 5:30-6:00 / HR<138；任一不达 → 休息 + 30 min 骑行 + 拉伸；HR>140 立即走跑结合。

**周日 5/17 — 长距离 25K ⭐⭐ 本周核心课**：6:00-6:30 出发；路线**世博滨江/徐汇滨江**（与 5/10 同路线对照漂移）；稳态 25K，**前 23K 严格 5:10-5:25 / HR<145，末 2K 不得加速**。
- **配速分档**：<20°C 取 5:05-5:15；20-25°C 取 5:15-5:25；>25°C 纯心率（<145），5:25-5:35
- **末段纪律**（本周第 1 KPI）：km24/25 配速绝不快于 km20-22，HR 不破 150 上限。**冲尾段就是 30km+ 崩盘的提前演练**——5/10 长距离教训不能再犯
- **补给**：10K + 18K 各 1 个能量胶（GU/SiS），每 5K 喝水
- **HR 纪律**：5/10/15/20/25K 主动看 HR 记录；末 5K HR - 首 5K HR ≤ 7 bpm 为合格

> P1W2 之所以加到 25K（macro P1W2 是 25K）：心血管耐受已被 P1W1 的 22K 长距离证明；瓶颈不是距离，是 HR 纪律。25K 是 P1W3 的 28K 的台阶。
