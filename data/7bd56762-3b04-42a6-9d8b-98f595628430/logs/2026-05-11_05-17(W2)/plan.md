# W2 训练计划 — 2026-05-11 ~ 05-17（Phase 1 W2/6，**结构修正周**）

**所在阶段**：Phase 1 基础有氧重建期 W2/6（距西马 22 周）
**周量目标**：65 km（不是计划原本 82km；因 W1 实际 70.3km / TL 1179 已超原计划负荷）
**核心任务**：把 W1 没做的 MP 段做了（**降级 4km**），把周六 22km LSD 补回，**强制 Z2 占比 ≥65%**
**纪律红线**：本周禁 race-effort、禁 VO2max 间歇、禁周二+周六双高强度。HR 145 报警，超即降速。

**周量小计**：5+12+11+11+0+22+4 = **65 km**

> **跑量为何反降**：W1 计划 75 → W1 实际 70.3km / TL 1179（5/10 11km Z2 补到接近周量但 TL 仍超 1108）。W2 计划 82 太激进。65km @ Z2 主导 = TL 估 750-800，让 CTI 在合理范围爬升。

### 周三 MP 段细则（4km 降级版）

W1 跳过了 6km MP，所以 W2 起步先做 4km（计划原 W1-W2 起点版）。**热身** 3km Z2 渐进到 4:50/km + 末 4×100m strides。**MP 核心** 4km @ 3:58-4:02（HR 起 155→中段 160-165→末 168 自然漂移）；HR >170 持续 1min 即降到 4:05 主动收尾。**降级触发**：5/12 晨 RHR ≥48 → 缩为 3km MP；睡眠 <6h → 改为 8km Z2。**收尾** 4km Z2 @ 5:00-5:10/km。

### 周六长跑细则（22km）

跑前 60min 燕麦 60g + 蜂蜜 + 香蕉 ≈70g 碳水。**清晨 5:30-6:30 出发**（避开热）。起步 5:30，10km 后 5:00-5:15。HR 前 18km 严格 ≤145，末 4km 自然漂 ≤152。**绝对不做 finish kick**。携 500ml 水 + 1 胶 @75min。跑后 30min 80g 碳 + 30g 蛋。

### 关键纪律 & 弹性

1. **本周完全禁项**：10K+ TT、track 间歇、Z4+ tempo（无论自我感觉如何）
2. 周二 / 四 / 六（前 18km）/ 日 严格 HR ≤145，超即减速
3. 高温 >25°C 时配速 +5-10s/km，HR 不变
4. 胫骨晨痛 ≥3/10 → 当日改椭圆机；≥4/10 → 完全休息
5. 长跑遇雨顺延周日；周三状态差（RHR >50 或睡眠 <5h）改 8km Z2

## 四、睡眠 + 监控（W2 硬指标）

W1 平均睡眠 6.9h，连续 5 天 <7.5h — **W2 必须打破这个模式**。每晚 22:30 上床，目标 6:30 起床 = 8h。14:00 后不喝咖啡/浓茶。睡前 30min 手机出卧室 + 5min 拉伸 + 4-7-8 呼吸。

| 信号 | 阈值 | 触发动作 |
|------|------|----------|
| 连续 3 晚睡眠 <6.5h | 红线 | 强制删周三 MP，周三改为 8km Z2 |
| 5/12 晨 RHR ≥48 | 黄灯 | 周三 MP 缩为 3km |
| 同周 ≥2 天 RHR >48 | 红灯 | 周日跑改完全休息 |
| 胫骨晨痛 ≥3/10 | 黄灯 | 当日改椭圆机 |
| 胫骨晨痛 ≥4/10 | 红灯 | 完全休息 |
| HRV 周降 ≥10% | 黄灯 | 周三 MP 缩 / 改 Z2 |

每日必记到 feedback.md：**RHR**、**HRV** 夜均、**体重**（仅周一）、**胫骨/脚踝晨痛**（0-10）。每次跑后立即 Garmin sport_note：RPE + 完成度 + 不适。
