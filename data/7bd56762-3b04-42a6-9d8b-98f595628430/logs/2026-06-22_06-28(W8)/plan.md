### 周三强度课（W8 锚点课，清晨 5:30）

**结构**：WU 2.5K @ 5:10→4:30 + 4×100m strides → **4K @ 半马配速** → **休息 5min（慢跑或走）** → **2×2K @ HMP-5~10s / 组间慢跑 3min** → **5×400m / 组间慢跑 60s** → CD 1.5K @ 5:20

> **设计理由**：以真实半马能力（推算 HM 1:24:47 = 4:01/km，阈值 3:46、LTHR 170）锚定的三段递进刺激——4K HMP 先把比赛配速糖原经济性铺开；2×2K 略快于 HMP（进 cruise/Z4）拉阈值；5×400m 收尾做神经募集与跑姿弹性。**主质量量 4+4+2 = 10K，整节 ~16K**。400m 段控制单次冲击（脚踝/胫骨敏感型），组数从 W7 的更长结构收到 5 组。

**① 4K 半马配速段**：
- 配速 **3:58-4:02/km**（HM 推算 4:01）；HR 锚 **162-170**，封顶 172
- 凉爽天（<22°C）按配速；**>25°C 心率优先**，配速放 4:02-4:10，HR 不破 172 即可

**② 2×2K（HMP-5~10s，cruise/Z4）**：
- 配速 **3:48-3:55/km**；HR 168-175，封顶 177
- 组间 **3min 慢跑**（4:50-5:10），HR 回到 ~155-160 再起第 2 组
- >25°C：放 3:53-4:00，心率优先（HR ≤177）

**③ 5×400m 间歇**：
- 配速 **3:28-3:36/km**（每个 83-86s）；地板 3:40（不准更慢），天花板 3:24（跑进即收当段，别拼快）
- 组间 **60s 慢跑**，HR 回到 ~160 再起下一组
- HR 自然到 175-185，**不设单段 cap**；安全红线 HR ≥187 持续 >15s → 当组结束
- >25°C：400m 放 3:34-3:42，心率/RPE 优先

**Abort（任 1 即停）**：
- 4K HMP 段普遍跑不进 4:02（凉爽天）或 HR 已破 172 → 缩到 3K，2K 段放 3:53-3:58
- 任一 2K 段配速漂 >3:58 持续 → 第 2 个 2K 改 1.5K 或并入慢跑
- 第 1-2 个 400m 跑不进 3:40 → 改 4 组
- 任一 400m 跑进 3:24 失控 → 收当段配速，不追更快
- AnE 课后 >3.5 / RPE >9 / 胫骨晨痛 ≥3/10 → 取消改椭圆 45min

**调整触发**：
- 6/23 晚 sleep <6h → 2K 段改 1 组、400m 降到 4 组；累计 6/22-6/23 <12h → 整节降为 4K HMP + 4×400m
- 6/24 晨 RHR ≥48 → 取消强度课，整课改 13K Z2
- T >28°C → 4K HMP 心率目标执行、2K 降 1 组、400m 降到 4 组放 3:36-3:44（心率/RPE 优先）

**执行流程**：5:00 起 / 5:15 50g 碳水 / 5:30 出门 / 5:50 起 4K HMP / 6:06 收、休 5min / 6:11 起 2×2K / 6:35 收、过渡 / 6:38 起 400m / 6:50 末组 / 6:55 CD / 7:10 30g 蛋白+60g 碳水。**当日不下肢力量、不踢球、不打乒乓。**
