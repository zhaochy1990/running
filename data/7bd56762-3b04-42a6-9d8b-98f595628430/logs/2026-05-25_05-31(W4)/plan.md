# W4 训练计划 — 2026-05-25 ~ 05-31（Phase 1 W4/6，减量周）

**阶段**：Phase 1 W4/6，距西马 21 周 | **周量**：~62 km（master plan = 70% deload）
**主题**：本周质量课为 **周三 20min 节奏跑 + 10×400m 间歇**（清晨 5:30）。其余 Z2 + 周六 LSD 25K + 完全休息日 ×2 缓解 W3 累积负荷。

> **本周纪律调整**：W3 7 天 readiness_gate 全周 red/yellow（持续 sleep_debt），L4 复合 94.85→94.02（-0.83）由 recovery 69.78→57.18 + economy 79→77 拉动；周六 21K LSD HR 全程漂到 145+（计划 ≤145 全程，实际只前 4K 达标），周二 800m × 12 完成但最后 2 rep 跑进 3:21/3:13（破 3:22 红线）。**W4 第一任务是把 recovery 拉回 70+、RHR 稳定 ≤45**，强度课压在周三，周量 60 km；砍 5/25 拖腿 5K 以把单次长距离从 18K 拉到 **25K**（≈2h05min，更接近 marathon-specific endurance 刺激）。

### 周三质量课（**W4 锚点课**，清晨 5:30）

**结构**：WU 3K @ 5:00→4:30 + 4×100m strides → **20min @ 3:48-3:54（Tempo / LT）** → **4min jog 恢复** → **10×(400m @ 3:25-3:35 + 90s rest jog)** → CD 2K @ 5:20

#### 节奏段（20min Tempo）

- 配速地板：3:54（LT 上限）；天花板：3:48（不快于）
- HR 目标：168-175（落 LT 区间），>178 持续 30s → 当组结束 abort 转 CD
- 预计完成距离 5.0-5.2 km
- 节奏段后 **4min 站立或慢跑** 自由形式恢复（HR 自然回 140）

#### 400m 间歇

**Rep 数决策矩阵**：

| 5/27 晨状态 | Reps | 节奏区间 |
|------------|------|---------|
| 干净（睡眠 ≥7.5h + RHR ≤44 + 5/26 易跑 HR 守住 ≤142） | **10** | 前 3 reps 3:32-3:35 → 中 4 reps 3:28-3:32 → 末 3 reps 3:25-3:30 |
| 标准（睡眠 7-7.5h + RHR 45-47） | **10** | 前 4 reps 3:32-3:35 → 中 4 reps 3:28-3:32 → 末 2 reps 3:25-3:30 |
| 睡眠不足（睡眠 <7h / RHR ≥48 / 节奏段已 HR 漂 >175） | **8** | 全程 3:30-3:35，**不准末段提速** |

**配速纪律**：

- 地板 3:35（不准慢于） / 天花板 **3:25（绝对红线，跑进即 abort）**
- W3 教训：最后 2 reps 跑进 3:13 / 3:21 = 破了 race-effort 红线。本周 10 reps 全部锁定 3:25-3:35，**不可有任何 1 rep 跑进 3:24**
- 末 3 reps 配速漂 >3:38 持续 2 组 → 当组收

**恢复**：每组之间 90s rest jog @ 6:00-6:30；HR 自然回 145-155 即可。

**HR 策略**：不设单组 HR cap；安全红线 HR ≥186 持续 30s → 当组结束后视情况收。

**Abort**（任 1 即停）：节奏段 rep 1 跑不进 3:58 / 400m rep 1 跑不进 3:38 / AnE >3.0 / AE >5.0 / RPE >9 / 任 1 rep 跑进 3:24 / 末 3 rep 配速漂 >3:38 持续 2 组。

**调整触发**：5/26 晚 sleep <7h → 8 reps + 关上限 3:30-3:35；5/25-5/26 累计 <13h → 仅做节奏段 + 改 6K Z2；5/27 晨 RHR ≥48 → 取消 400m 仅做节奏 + CD；胫骨晨痛 ≥3 → 改椭圆 45min。

**执行**：5:00 起 / 5:15 50g 碳水 / 5:30 出门 / 5:55 节奏 / 6:25 间歇 / 7:00 cooldown。**当日不下肢力量，不踢足球，不打乒乓**。

### 周六 LSD 25K 纯 Z2（master plan deload 长跑扩长）

> 把单次长距离从 master plan deload 默认 18K 拉到 25K，作为本周 marathon-specific endurance 主刺激。HR 纪律比距离更重要 —— 全程 ≤145，漂出去就减速。

**结构（25K，目标 ~2h05min）**：
- 前 8K：5:20→5:00 渐进入区间，HR 130-140
- 中 12K：5:00-5:10 稳态，HR 138-145
- 末 5K：5:00→5:30 cooldown，HR 漂上 cap → 减速到走/慢跑

**HR 145 报警 → 立即减速或走步**；末 5K HR 漂出 145 cap 不算红线，但触发减速。

**补给**：跑前 60min 50g 碳水；携 750ml 水 + 2 胶（@10K + @18K 处）。

**Abort**：HR 漂 >150 持续 1min（非上坡）→ 停下走 1min 再起跑；HR >155 持续 30s → 停跑改走；胫骨/脚踝不适 → 当场停。

**调整触发**：5/29 晨 RHR ≥47 → 距离砍到 22K；连续 2 天 RHR ≥48 → 砍到 20K；当日 >25°C → 配速 +5-10s/km 全程；当日 ≥27°C 或 humidity ≥85% → 砍到 20K 或推到周日（5/31 力量 B 取消）。

### 关键纪律

1. **本周禁**：MP / Tempo 长段（≥10min 除周三指定）/ 长间歇 / mile reps / 足球 / 乒乓 / 傍晚 18:00+ 操场训练
2. 配速显示：周三看配速；其余全部**只看 HR**（关闭配速显示）
3. **质量课 ±24h 不下肢力量**（力量 A 在周一晚；力量 B 在周日晨跑后）
4. >25°C 配速 +5-10s/km；400m 配速 +3-5s/km
5. 胫骨晨痛：≥3 椭圆 / ≥4 完全休息
6. 5/31 recovery：如 5/30 后 RHR ≥47 或 DOMS ≥4/10 → 改 5K 走 + mobility 30min
