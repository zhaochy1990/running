## 本周训练重点
本周核心是 **周三的强度课** 和 **周六的长距离课**，这两课请优先保证完成。其它训练（有氧跑、力量、灵活性）可根据自身反应随时调整：有氧跑可增减距离，疲劳时缩短或改休息——恢复也是训练的一部分，给自己留出恢复空间。

**阶段**：Phase 2 W3（力量耐力建设期，距西马 15 周）| **目标周量**：~80-86 km
**Form 分布期望**：提升期 45% + 维持期 35% + 比赛就绪 20%（build 阶段：chronic（cti）≈1116「OPTIMAL」→ 维持需周 dose ≈ chronic×7；本周做"质升量稳"，acute 持续略高于 chronic 驱动提升，**每天有 dose 避免 spike+flat**；下周 W10 进减量周 + 7/12 半马测试，本周是 build 收尾，**保质不冒量**）
**主题**：周三 **清晨凉爽窗口的阈值课（4K tempo → 休 4min → 4×1K cruise → 4×400m，软地面/塑胶道）**——这是本周的**核心诊断课**：他近况 Garmin 预测从 2:48 滑到 2:50-2:55、VO2max 60→58，但跑量、慢性负荷、HRV、RHR 全部强健（见下「本周复盘」），判定为**夏季高温把 pace:HR 效率压低导致的 VO2max 估算下滑**，不是真实掉体能。所以本周不靠堆量，而靠一节**移到 5:00-6:30 清晨凉爽窗口**的高质量阈值课，把上周被高温（94% 湿度傍晚操场）稀释的顶端能力在凉爽条件下重新跑出来 + **验证真实体能**。周六 **24-26K 渐加速长跑**（本人上周三已自练 25K，避免背靠背超量，本周长跑取下沿）。**山坡跑继续顺延**——本人脚踝/胫骨敏感型，强度课锁定软地面 tempo+cruise，不上山坡/不上最大 VO2max 间歇。

### 周三强度课（W9 锚点 / 体能诊断课，清晨 5:30 凉爽窗口，软地面/塑胶道）

**结构**：WU 2.5K @ 5:10→4:30 + 4×100m strides → **4K @ tempo（Z4）** → **休息 4min（慢跑或走）** → **4×1K @ cruise（HMP-5~10s）/ 组间慢跑 90s** → **4×400m / 组间慢跑 60s** → CD 1.5K @ 5:20

> **设计理由（本周核心）**：上周的傍晚阈值课在 **94% 湿度**下同 3:55 配速 HR 已顶到 170+，Garmin VO2max 估算被高温压到 58。本周把同类阈值刺激**移到 5:30 凉爽窗口**——目的有二：①在公平温度下**重测真实顶端**（凉爽天同 3:50-3:55 配速 HR 该回落到 168-173，若如此即证实"没掉体能、是高温 sag"）；②为 **7/12 半马测试**打阈值底子。4K 连续 tempo 拉 LT，4×1K cruise 巩固阈值耐受，4×400m 收尾做神经募集与跑姿弹性。**结构距离 = WU 2.9K + 主质量 9.6K（4+4+1.6）+ CD 1.5K = 14.0K**；时间制恢复慢跑（240s + 4×90s + 4×60s ≈ 840s）另落地约 2K，**整节落地 ~16K**。**伤病防护（HARD）**：本课锁定**软地面/塑胶跑道**，**不上山坡、不做最大 VO2max 间歇**；400m 控制单次冲击，组数收到 4 组，间歇后 7/02 强制软地面恢复跑。

**① 4K tempo 段（Z4）**：
- 配速 **3:48-3:55/km**；HR 锚 **168-175**，封顶 177
- 凉爽天（<22°C）按配速；**>25°C 心率优先**，配速放 3:53-4:00，HR 不破 177 即可
- **诊断读数**：凉爽窗口里若 3:50-3:55 配速 HR 稳在 168-173（低于上周傍晚同配速的 170-174）→ 证实高温 sag 假设，体能在线

**② 4×1K cruise（HMP-5~10s）**：
- 配速 **3:48-3:55/km**；HR 168-175，封顶 177
- 组间 **90s 慢跑**（4:50-5:10），HR 回到 ~158-162 再起下一组
- >25°C：放 3:53-4:00，心率优先（HR ≤177）

**③ 4×400m 间歇**：
- 配速 **3:28-3:36/km**（每个 83-86s）；地板 3:40（不准更慢），天花板 3:24（跑进即收当段，别拼快）
- 组间 **60s 慢跑**，HR 回到 ~160 再起下一组
- HR 自然到 175-185，**不设单段 cap**；安全红线 HR ≥187 持续 >15s → 当组结束
- >25°C：400m 放 3:34-3:42，心率/RPE 优先

**Abort（任 1 即停）**：
- 4K tempo 段普遍跑不进 3:55（凉爽天）或 HR 已破 177 → 缩到 3K，1K 段放 3:53-3:58
- 任一 1K 段配速漂 >3:58 持续 → 收到 3 组
- 第 1-2 个 400m 跑不进 3:40 → 改 3 组
- 任一 400m 跑进 3:24 失控 → 收当段配速，不追更快
- AnE 课后 >3.5 / RPE >9 / 胫骨晨痛 ≥3/10 → 取消改椭圆 45min

**调整触发**：
- 6/30 晚 sleep <6h → 1K 段改 3 组、400m 降到 3 组；累计 6/29-6/30 <12h → 整节降为 4K tempo + 4×400m
- 7/01 晨 RHR ≥48 → 取消强度课，整课改 13K Z2
- T >28°C（清晨已破 28 罕见，但若起步晚到日出后）→ 4K tempo 心率目标执行、1K 降 1 组、400m 降到 3 组放 3:36-3:44（心率/RPE 优先）

**执行流程**：5:00 起 / 5:15 50g 碳水 / 5:30 出门 / 5:50 起 4K tempo / 6:06 收、休 4min / 6:10 起 4×1K / 6:32 收、过渡 / 6:35 起 400m / 6:46 末组 / 6:50 CD / 7:05 30g 蛋白+60g 碳水。**当日不下肢力量、不踢球、不打乒乓。**
