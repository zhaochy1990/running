# W1 训练计划 — 2026-05-04 ~ 05-10（Phase 1 起点周）

**所在阶段**：Phase 1 基础有氧重建期 W1/6
**周量目标**：75 km
**核心任务**：把"真低强度"（HR ≤ 145）的跑量占比拉到 50%+，**首次引入 MP 段（4km @ 3:58-4:02）建立比赛配速肌肉记忆**
**纪律红线**：所有标 Z2 的跑步 → **手表只显示心率和时间，关闭配速显示**；HR 上限报警 145 bpm

**周量小计**：5 + 13 + 12 + 12 + 0 + 22 + 11 = **75 km** ✓

### 周三 MP 段执行细则

> 这是本周最重要的一课。MP（Marathon Pace）= 3:58-4:02/km。

**热身（3km Z2，约 15min）**：从 5:30/km 起步，渐进到 4:50/km。最后 800m 加 4×100m strides（步频高、肌肉激活，但不算冲刺）。

**MP 核心段（4km @ 3:58-4:02，约 16min）**：
- 配速波动控制在 ±3s/km 内（即 3:55-4:05 都算合格）
- HR 起步 155，1km 内爬到 160-165，全段稳定 160-168
- **若 HR > 170 持续 1min → 立即降到 4:05-4:10，本次 MP 段提前结束**
- 节奏感觉：呼吸 3:3 或 2:2，能说 1-2 个词的话但不能成句

**收尾（5km Z2，约 25min）**：MP 段结束后立刻减速到 5:00/km，HR 自然回落到 ≤ 145，Z2 跑回。

**总时长**：约 50-55 分钟。

### 周六长跑执行细则

- **起跑前 60min**：燕麦 60g + 蜂蜜 + 1 根香蕉 ≈ 70g 碳水
- **配速**：起步 5:20/km，10km 后稳定 5:00-5:10/km；**不在长跑中加速**（W1 不带 finish kick）
- **HR 控制**：前 15km 严格 ≤ 145；末 7km 允许漂移到 ≤ 152（这是体温/疲劳的正常生理反应，不是加速）
- **补给**：携带 500ml 水 + 1 个能量胶；**第 75 分钟（约 15km 处）补 1 个胶**
- **跑后 30min**：碳水 80g + 蛋白 30g（米饭 + 鸡胸 + 蔬菜，或蛋白奶昔 + 香蕉 + 吐司）

### 关键纪律

1. 周二/周四/周日 + 周六前 15km：HR 上限 **145**，超出立即减速
2. 周三 MP 段：HR 上限 **170**，超出降配速 5-10s/km，本次 quality 不强求
3. 长跑日不可改成 sub-MP fartlek；周六做 Z2 就是 Z2
4. **若胫骨晨痛 ≥ 3/10**：当日改椭圆机 30-45min；≥ 4/10：整日休息

### 跑前热身标准（每日 5min）

| 动作 | 时长 |
|------|------|
| 慢走 + 高抬腿 | 1min |
| 弓步走 | 10 步/侧 |
| 踢臀跑 | 30s |
| 前后/侧向 leg swing | 10 次/侧 × 2 方向 |
| 髋部 90/90 转动 | 5 次/侧 |
| 踝关节绕环 | 10 圈/侧 |

> 周三 MP 段额外加 4×100m strides（在热身末段）。

### 天气/状态弹性

- **长跑遇雨**（周六）：可顺延到周日，原周日恢复跑改到周一
- **周三 MP 段当天身体不在状态**（RHR > 50，或睡眠 < 5h）：降级为 12km Z2
- **气温 > 28°C**（西安/华东 5 月偶有）：MP 段配速 +5-10s（即 4:03-4:10），HR 不变

## 二、力量训练

### 每日预防 Routine（5-10 分钟，跑前/睡前任选）

| 动作 | 组数 × 次数 | 时机 |
|------|-------------|------|
| 踝内翻弹力带 | 3×12/侧 | 跑前 |
| 踝外翻弹力带 | 3×12/侧 | 跑前 |
| 胫骨前肌抬脚（坐姿勾脚） | 3×15 | 跑前 |
| 单腿提踵（慢下 3s） | 3×10/侧 | 跑后 |
| 小腿前侧 + 腓肠肌筋膜放松 | 30s/侧 | 跑后 |

### 力量 A — 下肢稳定性（周一 5/4，40-50 分钟，跑前进行）

| 动作 | 组数 × 次数 | 重点 |
|------|-------------|------|
| 单腿离心提踵（台阶，慢下 3s） | 3 × 8/侧 | 跟腱+胫骨偏心负荷 |
| BOSU 球（或软垫）单腿静站 | 3 × 30s/侧 | 踝关节稳定 |
| 弹力带绕踝侧向行走 | 3 × 10步 × 2方向 | 踝外侧 + 臀中肌 |
| 哑铃保加利亚分腿蹲 | 3 × 8/侧（**首次起：8-12kg/手**） | 单腿股四 + 平衡 |
| 单侧罗马尼亚硬拉 | 3 × 8/侧（**首次起：12-16kg/手**） | 后链 + 单腿稳定 |
| 蚌式开合（弹力带） | 3 × 15/侧 | 臀中肌 |
| 单腿臀桥（脚跟踩瑜伽垫） | 3 × 12/侧 | 臀大肌 |
| 平板支撑 + 交替抬腿 | 3 × 30s | 核心抗旋 |

> W1 是基线建立周：所有动作严格记录完成度（重量、组数、技术不变形的最大次数），下周递增。

## 三、营养策略（68kg，TDEE ~3000-3500 kcal/天）

### 宏量基线

| 类型 | 碳水 | 蛋白 | 脂肪 | 总热量 |
|------|------|------|------|--------|
| 普通训练日（周二/四/日） | 5-6 g/kg = **340-410g** | 1.6 g/kg = 109g | 1.0 g/kg = 68g | ~3000 kcal |
| 长跑/质量课日（周三/六） | 7-8 g/kg = **475-545g** | 1.8 g/kg = 122g | 1.0 g/kg = 68g | **3300-3600 kcal** |
| 完全休息日（周五） | 4-5 g/kg = **270-340g** | 1.6 g/kg = 109g | 1.0 g/kg = 68g | ~2800 kcal |

### 训练补给

- **晨跑前**（< 60 分钟 Z2）：可空腹或 1 根香蕉
- **周三 MP 段前**（提前 60min）：吐司 1 片 + 蜂蜜 + 1 根香蕉（约 50g 碳水）
- **周六长跑**：起跑前 60min 燕麦 + 蜂蜜 + 1 根香蕉（70g 碳水）；**跑中第 75 分钟（≈15km 处）1 个能量胶**；500ml 水
- **跑后 30min 黄金窗口**：碳水 60-80g + 蛋白 25-30g

### 实操参考菜单

**普通日（周二）**
- 早：燕麦 80g + 牛奶 250ml + 香蕉 1 根 + 鸡蛋 2 个（碳水 ~90g，蛋白 ~25g）
- 午：糙米饭 200g（生重 80g）+ 鸡胸 150g + 蔬菜一大碗 + 油 10g（碳水 ~80g，蛋白 ~40g）
- 晚：意面 100g（生重）+ 牛肉 120g + 番茄汁 + 蔬菜（碳水 ~75g，蛋白 ~35g）

**长跑日（周六，多 +130-200g 碳水）**
- 早（跑前 60min）：燕麦 60g + 蜂蜜 30g + 香蕉 1 根
- 跑中：能量胶 1 个（30g 碳水）@ 75min
- 跑后立即：蛋白奶昔 30g + 香蕉 + 吐司 2 片 + 蜂蜜
- 午：糙米饭 250g（生重 100g）+ 鸡胸 / 牛肉 150g + 蔬菜
- 加餐：水果 + 酸奶
- 晚：饺子或饭 + 鱼 / 蛋

### 体重监控

- 每周一晨起空腹称重 1 次，写入 feedback.md
- W1 起点目标：维持 68 kg（不减不增）。**不追求减重**——BMI 21.5 已是马拉松理想体型。

## 四、睡眠与恢复（本周硬指标）

> 数据现状：日均 6.5h，深睡 64min，REM 62min。**当前最大瓶颈**。

| 项目 | 本周目标 |
|------|----------|
| 每晚就寝时间 | ≤ 22:30（最迟 23:00） |
| 每晚总睡眠 | ≥ 7.5h（目标 8h） |
| 14:00 后 | 不喝咖啡 / 浓茶 |
| 睡前 30min | 手机放卧室外，做 5min 拉伸 + 4-7-8 呼吸 |
| 周日复盘 | 写入 feedback.md：本周睡眠均时、Garmin 睡眠分均值 |

**红线**：连续 3 晚 < 6h → 自动减一次质量课（删周三 MP 段）。

## 五、监控指标（每日必记）

每天早晨在 feedback.md 记录：

- **静息心率**（Garmin 自动）：基线 41-46，> 50 持续 3 天 = 红灯
- **HRV 夜均**（Garmin）：基线 93-129
- **体重**（仅周一记一次）
- **胫骨/脚踝晨痛评分**（0-10）：≥ 3 → 当日改交叉训练；≥ 4 → 跳过强度课

每次跑后立即在 Garmin 写 sport_note：
- 主观 RPE（1-10）
- 完成度（按计划 / 提前结束 / 改强度）
- 身体不适（如有）

| 检查项 | 通过标准 |
|--------|----------|
| 周二 Z2 平均心率 | ≤ 140 |
| **周三 MP 段 4km** | **配速 3:58-4:02，HR 全程 ≤ 168，无掉速** |
| 周六长跑 22km | 前 15km HR ≤ 145，末 7km ≤ 152，无胫骨/脚踝痛 |

> **MP 段是 Phase 1 最早的目标配速试金石**：
> - 4km @ 3:58-4:02 完成且 HR ≤ 168 → Phase 1 起步顺利，下周可加距离到 5-6km
> - HR 漂到 170+ 才能保住配速 → 说明心血管成本仍偏高，W2 不加距离仍 4km，让 HR 降下来
> - 配速保不住（掉到 4:05+） → W2 改用 2×2km @ MP / rest 90s 切段做
