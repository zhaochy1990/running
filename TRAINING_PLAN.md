## Monitoring & Adjustment Triggers

Track these weekly via `python -m coros_sync sync && python -m coros_sync analyze`:

| Metric | Green | Yellow | Red |
|--------|-------|--------|-----|
| Weekly km | 80-100 | 65-80 | < 65 |
| Easy run HR | < 145 | 145-152 | > 152 |
| RHR | < 50 | 50-55 | > 55 (overtraining) |
| HRV | Stable/rising | -10% drop | -20% drop (back off) |
| Z1-Z2 % | > 75% | 65-75% | < 65% (too hard) |
| VO2max | > 59 | 57-59 | < 57 (losing fitness) |

**Adjustment rules**:
- 2+ red metrics in same week → cut volume 30% for 1 week
- HRV drops 20%+ → skip quality session, easy run instead
- RHR elevated 5+ bpm for 3+ days → rest day
- Illness/injury → skip Phase, don't compress. Health first.
