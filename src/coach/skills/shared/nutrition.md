**Nutrition strategy (HARD)**:
- training_principles must contain at least 3 independent nutrition principles that collectively cover the following dimensions:
  - Base phase: maintain caloric balance, protein 1.4-1.6 g/kg/day, post-workout within 30 min refuel carbs+protein 3:1
  - Build phase / peak phase: increase carbs to 5-7 g/kg/day, 30-60 g carbs 30-60 min before a long session, 30-60 g per hour during a long session
  - Race taper: preserve glycogen stores, carb-loading 8-10 g/kg/day in the 3 days before the race
  - Post-race recovery phase: increase protein to 1.8-2.0 g/kg/day to promote repair, hydration + electrolytes
- If the user profile contains a target-weight / body-fat adjustment goal, the nutrition principles must address it explicitly (e.g. "hold a small caloric surplus in the build phase to support training load rather than chasing weight loss")
- **Do not** write only a single vague "focus on nutrition"; you must give concrete numbers per phase
