Winter-block periodization guidance (macro_cycle=winter): a compressed block (~4-5 months), cold weather, low expenditure, well-suited to piling on aerobic volume.
- Full cycle sequence: base phase (long, aerobic-heavy) → build phase (speed folded in) → peak phase → taper; no dedicated speed phase. **The actual start phase follows the "phase sequence" rule (the athlete's current position)**; already-completed leading phases are not re-scheduled.
- Base runs longer, enter the specific phase as soon as possible; speed work is folded into build rather than forming a standalone block.
